# 2주차 — HTML과 CSS (단계별 실습 1~5)

하나의 개인 홈페이지를 실습 1부터 5까지 순서대로 발전시킨다.
각 lab 폴더는 해당 단계까지 완성된 모습이다. **이전 단계 코드를 복사해서 이어서 작업할 것.**

| 실습 | 폴더 | 배우는 것 |
|---|---|---|
| 실습 1 | `lab1/` | HTML 골격과 기본 태그 (h, p, img, a, ul/ol) |
| 실습 2 | `lab2/` | 시맨틱 태그, 테이블, 폼 |
| 실습 3 | `lab3/` | CSS 분리, 선택자, 박스 모델, 중앙 정렬 |
| 실습 4 | `lab4/` | Flexbox 내비게이션과 카드 레이아웃 |
| 실습 5 | `lab5/` | 미디어 쿼리로 반응형 완성 (최종 제출물) |

## 확인 방법
각 폴더의 `index.html`을 Live Server로 실행한다.
실습 5는 개발자 도구(F12) → 기기 모드(Ctrl+Shift+M)에서 모바일 화면도 확인한다.
