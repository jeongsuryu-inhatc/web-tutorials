# 6주차 — JavaScript 미니 프로젝트: 브라우저 Todo 앱

지금까지 배운 것을 모아 Todo 앱을 완성한다. 핵심은 **배열 상태(todos)와 화면 그리기(render)의 분리**다.

| 실습 | 파일 | 단계 |
|---|---|---|
| 실습 1 | `lab1.html` | 화면 마크업과 스타일 (기능 없음) |
| 실습 2 | `lab2.html` | 배열 상태 + render 함수 분리 |
| 실습 3 | `lab3.html` | 추가(버튼·Enter)와 삭제 |
| 실습 4 | `lab4.html` | 완료 체크 토글과 취소선 |
| 실습 5 | `lab5.html` | 남은 개수 표시 + 전체 삭제 (최종 제출물) |
