# MBTI 챗봇 디자인 명세 (Stitch 생성 예시)

> https://stitch.withgoogle.com/ 에서 프롬프트로 생성한 시안을 정리한 예시.
> 실제 실습에서는 본인이 Stitch로 만든 시안 내용으로 대체할 것.

## 사용한 Stitch 프롬프트

```
MBTI 판별 챗봇의 모바일 웹 채팅 화면.
상단에 앱 이름과 진행 상태(질문 3/6), 가운데는 말풍선 대화 영역
(봇은 왼쪽 회색, 사용자는 오른쪽 파란색), 하단에 입력창과 전송 버튼.
마지막 결과 화면에는 MBTI 4글자를 크게, 아래에 성향 설명 카드.
파란색 포인트의 밝고 미니멀한 디자인.
```

## 화면 구성

- **헤더**: 좌측 앱 이름(파란색 볼드), 우측 진행 상태 "질문 3/6" (회색 13px)
- **대화 영역**: 세로 스크롤. 봇 말풍선은 좌측 정렬·연회색(#F3F4F6), 사용자 말풍선은 우측 정렬·파랑(#2563EB)에 흰 글자. 모서리 radius 16px, 꼬리 쪽만 4px
- **입력 영역**: 하단 고정. 둥근 입력창 + 파란 알약형 전송 버튼
- **결과 표시**: 마지막 봇 말풍선에 MBTI 4글자와 축별(E/I, S/N, T/F, J/P) 근거 설명

## 색상

| 용도 | 색상 |
|---|---|
| 주조색 (버튼, 사용자 말풍선, 강조) | #2563EB |
| 배경 | #F9FAFB |
| 카드/봇 말풍선 | #F3F4F6 |
| 본문 글자 | #1F2937 |
| 보조 글자 | #6B7280 |

## 타이포그래피·기타

- 시스템 고딕 계열, 본문 14px, 줄간격 1.5
- 모바일 우선 (최대 폭 480px 중앙 정렬)
- 애니메이션·외부 라이브러리 없음
