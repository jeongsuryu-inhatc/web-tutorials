## 실행 환경

- 1~7주차: 브라우저 + Live Server (VS Code 확장)
- 8~11주차: Node.js — 각 폴더에서 `npm install` 후 `node <파일명>`
- 10~11주차: MySQL 필요 (`schema.sql` 먼저 실행, `db.js` 비밀번호 수정)
- 14주차: AWS 계정 필요

## 학습 원칙

1. 코드를 **직접 타이핑**한다 (복사·붙여넣기 최소화).
2. 각 주의 마지막 실습이 그 주의 **최종 제출물**이다.
3. 7주차부터 모든 결과물을 GitHub으로 관리한다.
4. AI를 쓸 때는 생성된 코드를 반드시 이해하고 검증한다 — 주도권은 나에게.
