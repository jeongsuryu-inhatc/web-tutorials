## 실습 1. 생성형 AI 사용 환경 만들기
1. Claude(https://claude.ai) 또는 ChatGPT(https://chat.openai.com)에 가입한다.
2. 자유 주제로 3번 이상 대화해 보고, 답변에서 좋았던 점/이상했던 점을 메모한다.

## 실습 2. 프롬프트 비교 실험
같은 목적의 질문을 **모호하게** 한 번, **구체적으로** 한 번 물어보고 결과를 비교한다.

| 구분 | 프롬프트 예시 |
|---|---|
| 모호 | "CSS 알려줘" |
| 구체 | "웹 초보 대학생에게 CSS 박스 모델을 그림 비유로 설명하고, margin과 padding의 차이를 코드 예시 하나로 보여줘" |

`prompt_report_template.md`에 비교 결과를 기록한다.

## 실습 3. AI에게 조사시키고 검증하기
AI에게 아래 사건들을 조사시키고, **검색으로 사실 여부를 직접 검증**한다 (AI는 틀릴 수 있다!).
- 1956 다트머스 회의 (AI라는 용어의 탄생)
- 머신러닝과 딥러닝의 차이
- 2016 이세돌 vs 알파고
- 2017 트랜스포머 논문 "Attention Is All You Need"
- LLM과 생성형 AI의 등장

## 실습 4. AI 역사 타임라인 웹페이지
검증한 자료로 타임라인 웹페이지를 만든다. **AI에게 코드 생성을 시키되, 결과를 직접 읽고 수정할 것.**
`timeline_example.html`이 완성 예시다.
