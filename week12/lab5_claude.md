# 실습 5 — Claude 설치와 특징 실습

## 5-1. 설치

1. 웹: https://claude.ai 접속 → 계정 생성 (무료 티어로 충분)
2. 데스크톱 앱(선택): https://claude.ai/download 에서 내려받아 설치 후 로그인
3. (선택·유료 플랜 필요) 터미널 CLI인 Claude Code:
   ```bash
   npm install -g @anthropic-ai/claude-code
   claude   # 프로젝트 폴더에서 실행
   ```

## 5-2. 특징 실습 ① — Artifacts로 웹페이지 즉시 미리보기

Claude에 아래를 입력하고, 오른쪽 Artifacts 창에서 **바로 실행되는 결과**를 확인한다.

```
5주차에 만든 방명록 페이지를 개선하고 싶어.
이름과 메시지를 입력하면 목록에 추가되는 방명록을 HTML 한 파일로 만들어줘.
조건: 외부 라이브러리 금지, 파스텔 톤의 단순한 디자인, 삭제 버튼 포함.
```

- 결과 코드를 내 `week5/lab5.html`과 비교해 보고, 다른 접근 방식 1가지를 찾아 메모한다.

## 5-3. 특징 실습 ② — 문서 이해

```
이 코드를 한 줄씩 초보자에게 설명해줘. SQL 인젝션 위험이 있는 부분이 있는지도 알려줘.
```
