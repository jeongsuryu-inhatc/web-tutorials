# 실습 6 — ChatGPT 설치와 특징 실습

## 6-1. 설치

1. 웹: https://chat.openai.com 접속 → 계정 생성 (무료 티어로 충분)
2. 데스크톱 앱(선택): https://openai.com/chatgpt/download 에서 설치 후 로그인
3. (선택·유료 플랜 필요) 터미널 CLI인 Codex:
   ```bash
   npm install -g @openai/codex
   codex   # 프로젝트 폴더에서 실행
   ```

## 6-2. 특징 실습 ① — 데이터 분석

1. 메모장으로 `scores.csv`를 만든다.
   ```csv
   name,score
   홍길동,85
   김철수,92
   이영희,78
   박민수,95
   최지우,63
   ```
2. ChatGPT에 파일을 업로드하고 요청한다.
   ```
   이 CSV를 분석해서 평균·최고점·최저점을 알려주고, 점수 분포를 막대 그래프로 그려줘.
   ```
3. 4주차 실습(lab3~4)에서 우리가 JavaScript로 직접 구현한 것과 결과를 비교한다.

## 6-3. 특징 실습 ② — 이미지 생성

```
내 Todo 웹 서비스의 로고를 만들어줘.
파란색 계열, 체크 표시 모티브, 단순하고 세련된 플랫 디자인, 배경 투명.
```

- 생성된 이미지를 저장해 두면 11주차 Todo 앱의 파비콘/로고로 쓸 수 있다.

## 6-4. 제출물

- 데이터 분석 결과(그래프) 스크린샷
- 생성한 로고 이미지 파일
