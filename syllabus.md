### 3.2 2주차 — HTML과 CSS

| 실습 | 실습 내용 | 결과물 |
|---|---|---|
| 실습 1 | **HTML 문서 구조와 기본 태그** — index.html에 `<!DOCTYPE html>`, html·head·body 골격 작성 → h1~h3 제목과 p 문단 3개로 자기소개 작성 → img로 프로필 사진 넣기 → a 태그로 좋아하는 사이트 3곳 링크 → ul/ol로 취미 목록 만들기 | 기본 태그로 구성된 자기소개 페이지 |
| 실습 2 | **시맨틱 태그와 폼·테이블** — 실습1 페이지를 header·nav·main·section·footer 구조로 재구성 → nav에 소개/시간표/연락처 3개 메뉴 링크 → table로 일주일 시간표 작성 → form에 이름(input)·이메일(input type=email)·메시지(textarea)·보내기(button) 구성 | 시맨틱 구조 페이지 + 시간표 + 연락처 폼 |
| 실습 3 | **CSS 기초 — 선택자와 박스 모델** — style.css를 분리해 link로 연결 → 배경·글자·포인트 3색 팔레트와 기본 폰트 지정 → 태그·클래스·id 선택자를 각각 1회 이상 사용 → margin·padding·border·border-radius로 소개 영역을 카드 박스로 만들기 → max-width와 `margin: 0 auto`로 본문 중앙 정렬 | CSS 분리·카드 박스 적용 페이지 |
| 실습 4 | **Flexbox 레이아웃** — nav를 `display: flex` 가로 메뉴로 변경(로고 왼쪽, 메뉴 오른쪽 — justify-content 활용) → 취미/프로젝트를 3개의 카드로 만들어 flex와 gap으로 가로 배치 → 카드에 hover 시 색이 바뀌는 트랜지션 추가 | 가로 내비게이션 바 + 3단 카드 레이아웃 |
| 실습 5 | **반응형 마무리** — `@media (max-width: 768px)`로 모바일에서 카드가 세로 1열, nav가 세로 메뉴로 바뀌게 처리 → 개발자 도구 기기 모드로 모바일 화면 확인 → 여백·정렬을 다듬어 최종 완성 | 반응형 개인 홈페이지 완성본(최종 제출물) |
